# Business Website Project

## Project Title
Responsive Business Website

## Description
This project is a responsive business website developed using HTML, CSS, JavaScript, and Bootstrap.  
The website includes multiple pages such as Home, About, Services, and Contact.

## Features
- Responsive Design
- Navigation Bar
- Hero Section
- About Page
- Services Page
- Contact Form with JavaScript Validation
- Footer Section
- Mobile Friendly Layout

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap

## How to Run the Project

1. Download or clone the project
2. Open the folder in VS Code
3. Install Live Server extension
4. Right click on index.html
5. Click "Open with Live Server"
